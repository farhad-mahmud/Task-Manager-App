# #️⃣ AnyTasks - Frontend UI

A minimal, modern, and beautifully responsive user interface built to manage tasks. It features a board divided into To Do, In Progress, and Done stages, along with device-based session isolation.

---

# 🛠️ Tech Stack & Features

* **🚀 Framework:** Next.js (App Router)
* **⚛️ Library:** React (v19)
* **🎨 Styling:** Tailwind CSS (v4)
* **📝 Language:** TypeScript
* **🔒 Task Isolation:** Uses browser `localStorage` to generate a unique session UUID, sending it via custom headers (`X-Session-ID`) to ensure device privacy.
* **📱 Design:** Fully responsive, clean layouts with modern typography and interface transitions.
